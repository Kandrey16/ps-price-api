насчет SKU
  страницы игр делаятся на маркетинговые и обычные. маркетинговые содержат в ссылке slug. в обычных такая возможность отсутствует. Обычные содержат __NEXT_DATA__, в которых есть необходимые мне данные.
  на данном этапе проблематично извлекать данные из обычных страниц. на раннем прототипе не буду использовать эти данные. 
  В дальнейшнем продвижении разработки расширим возможности, сделаю иные источники извлечения данных (API, другие страницы и т.д...)
